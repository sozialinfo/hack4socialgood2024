# Hack4SocialGood 2024

Am [Hack4SocialGood 2024](https://bd.hack4socialgood.ch/project/91) haben wir für [HEKS](https://www.heks.ch/) eine Challenge angenommen.

Die HEKS-Website richtet sich in erster Linie an Spender:innen. Neu sollen aber auch Projektteilnehmende in ihrer jeweiligen Sprache Informationen über HEKS-Projekte erhalten und sich bei den Projektverantworltichen mit Fragen oder Beschwerden melden können.

Dafür haben wir im Rahmen der Challenge eine Demo mit der No-Code Plattform [Bubble](https://bubble.io) erstellt.

Die **Demo** kann über diesen Link aufgerufen werden: https://heks.bubbleapps.io/
* Username: hack4socialgood2024
* Passwort: hack4socialgood2024

Der **Bubble-Editor** ist im "read-only" Modus ebenfalls öffentlich zugänglich: https://bubble.io/page?id=heks&tab=tabs-1
